# README
Procedure to run 
1. rails db:migrate RAILS_ENV=development
2. rake db:reset
3. rails s
In order to run this in localhost, enter rake db:reset first before entering rails s on command line

- ruby 2.3.0p0
- Rails 5.0.3\
